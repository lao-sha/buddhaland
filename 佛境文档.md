# BuddhaLand 佛境（前端重构方案：React + TypeScript + three.js）

本版本重构佛境前端技术栈为 React + TypeScript + three.js，彻底移除原方案中的 UE5 与 C++，以便更好地与 Substrate/Polkadot 生态集成，并兼顾跨平台与浏览器可访问性。

## 1. 目标与原则
- 浏览器可访问：无需本地原生客户端，直接通过 Web 端体验佛境。
- Web3 原生：与本链 Runtime Pallets（例如 Karma）无缝交互。
- 3D 沉浸体验：以 three.js 实现轻量级、可拓展的 3D 场景。
- 简洁可维护：React + TS 组件化、类型安全，提升迭代效率。

## 2. 总体架构

### 2.1 前端技术栈
- **核心框架**: React 18 + TypeScript 5.0+
- **3D 渲染**: Three.js + WebGL + React Three Fiber (R3F)
- **WebGL增强**: 
  - 原生 WebGL 2.0 API 用于高性能脑波可视化
  - GLSL 着色器实现实时粒子系统和光效
  - WebGL Compute Shaders 处理脑波数据分析
- **状态管理**: Zustand + React Query
- **UI 组件**: Ant Design / Material-UI
- **构建工具**: Vite + SWC
- **样式方案**: Tailwind CSS + Styled Components
- **脑波集成**: WebBluetooth API + NeuroJS

### 2.2 系统架构
- 前端：React + TypeScript + three.js + WebGL（可选：react-three-fiber 以提升 React 与 Three 的协作）
- 链交互：@polkadot/api 通过 WS 连接本地/远程节点
- 账户与签名：polkadot{.js} 浏览器扩展
- 后端/区块链：Substrate Runtime（已集成 Karma 等 Pallet）

## 3. 前端模块设计
- UI 层：账户连接、资产/Karma 展示、交互面板（签到、功德行为）
- 场景层：three.js 搭建主场景（光照、相机、控制器），基于 Karma/功德渲染粒子或场景元素强度
- 状态层：Zustand/Redux（推荐 Zustand）管理全局状态（账户、链状态、场景状态）
- 服务层：
  - polkadotApi：封装 WS 连接、查询与交易（如 Karma::daily_checkin）
  - storage 查询：KarmaBalances、TotalMeritValue 等

## 4. 数据上链与双Pallet激励机制

### 4.1 双Pallet架构设计
基于 [Polkadot SDK](https://paritytech.github.io/polkadot-sdk/master/polkadot_sdk_docs/polkadot_sdk/index.html) 最佳实践，采用职责分离的双pallet设计：

- **Meditation Pallet**：专门处理冥想会话数据验证与存储
  - 脑波数据验证、会话记录、防重复提交
  - 冥想质量评分、时长统计、连续性跟踪
- **Reward Pallet**：专门处理奖励计算与发放机制  
  - 基于冥想质量的动态奖励算法
  - 多维度奖励因子（深度、专注、连续性）
  - 与现有 Karma 系统的集成接口

### 4.2 WebGL增强的实时可视化
- **高性能渲染**：WebGL 2.0 原生API确保60fps流畅体验

- **粒子系统**：WebGL Compute Shaders驱动千万级粒子动画

## 10. 脑波驱动禅修冥想场景设计

### 10.1 脑波数据采集与处理

#### 硬件支持
- **推荐设备**: Muse头环、NeuroSky MindWave、Emotiv EPOC+
- **连接方式**: WebBluetooth API（浏览器原生支持）
- **数据频率**: 256Hz采样率，实时传输
- **脑波频段**:
  - Alpha波（8-12Hz）：放松专注状态
  - Beta波（13-30Hz）：清醒警觉状态  
  - Theta波（4-8Hz）：深度冥想状态
  - Delta波（0.5-4Hz）：深度放松状态
  - Gamma波（30-100Hz）：高度觉知状态

#### 数据处理管道
```typescript
/**
 * 脑波数据实时处理类
 * 负责从头环设备接收原始EEG信号，进行滤波、特征提取和状态识别
 */
class BrainwaveProcessor {
  private fftWindow: number[] = [];
  private meditationLevel: number = 0;
  private attentionLevel: number = 0;
  
  /**
   * 处理原始EEG数据并提取冥想特征
   * @param rawData 原始脑波数据数组
   * @returns 处理后的冥想状态指标
   */
  processRawEEG(rawData: number[]): MeditationMetrics {
    // FFT频谱分析
    const spectrum = this.performFFT(rawData);
    
    // 频段功率提取
    const alphaPower = this.extractBandPower(spectrum, 8, 12);
    const thetaPower = this.extractBandPower(spectrum, 4, 8);
    const betaPower = this.extractBandPower(spectrum, 13, 30);
    
    // 冥想深度计算 (Alpha/Beta比值 + Theta增强)
    const meditationDepth = (alphaPower + thetaPower * 1.5) / (betaPower + 1);
    
    // 专注程度计算
    const focusLevel = alphaPower / (alphaPower + betaPower);
    
    return {
      meditationDepth: Math.min(meditationDepth * 100, 100),
      focusLevel: Math.min(focusLevel * 100, 100),
      alphaPower,
      thetaPower,
      betaPower,
      timestamp: Date.now()
    };
  }
}
```

#### 防作弊机制
```rust
/**
 * 脑波数据真实性验证
 * 检测异常模式、静态数据、设备断连等作弊行为
 */
#[pallet::call]
impl<T: Config> Pallet<T> {
    pub fn submit_meditation_data(
        origin: OriginFor<T>,
        session_data: Vec<BrainwaveDataPoint>,
        session_summary: MeditationSession<T::Moment>,
    ) -> DispatchResult {
        let who = ensure_signed(origin)?;
        
        // 数据完整性验证
        ensure!(session_data.len() >= 300, Error::<T>::InsufficientData); // 至少5分钟数据
        
        // 异常检测
        let quality_score = Self::calculate_data_quality(&session_data)?;
        ensure!(quality_score >= 60, Error::<T>::LowQualityData);
        
        // 生物特征一致性检验
        let biometric_valid = Self::verify_biometric_signature(&who, &session_data)?;
        ensure!(biometric_valid, Error::<T>::BiometricMismatch);
        
        // 存储验证通过的数据
        let session_id = Self::next_session_id(&who);
        MeditationSessions::<T>::insert(&who, session_id, session_summary);
        
        Ok(())
    }
}
```

#### 实时视觉反馈系统
```typescript
/**
 * 脑波驱动的3D场景控制器
 * 根据实时脑波数据调整场景光照、粒子、音效等元素
 */
class BrainwaveSceneController {
  private scene: THREE.Scene;
  private lotusParticles: THREE.Points;
  private ambientLight: THREE.AmbientLight;
  private spotLight: THREE.SpotLight;
  
  /**
   * 根据冥想深度更新场景视觉效果
   * @param metrics 脑波处理后的冥想指标
   */
  updateSceneFromBrainwaves(metrics: MeditationMetrics): void {
    const { meditationDepth, focusLevel, alphaPower, thetaPower } = metrics;
    
    // 光照强度映射 (Alpha波越强，环境越明亮)
    const lightIntensity = 0.3 + (alphaPower / 100) * 0.7;
    this.ambientLight.intensity = lightIntensity;
    
    // 聚光灯颜色温度 (Theta波影响色温)
    const warmth = thetaPower / 100;
    const lightColor = new THREE.Color().lerpColors(
      new THREE.Color(0x87CEEB), // 冷蓝色
      new THREE.Color(0xFFD700), // 暖金色
      warmth
    );
    this.spotLight.color = lightColor;
    
    // 莲花粒子数量和运动速度
    const particleCount = Math.floor(1000 + meditationDepth * 30);
    this.updateLotusParticles(particleCount, focusLevel);
    
    // 场景雾化效果 (专注度越高雾越少)
    const fogDensity = Math.max(0.001, 0.01 - (focusLevel / 100) * 0.008);
    this.scene.fog = new THREE.Fog(0xcccccc, 10, 200 * (1 + fogDensity));
  }
  
  /**
   * 更新莲花粒子系统
   * @param count 粒子数量
   * @param focusLevel 专注程度影响粒子聚合度
   */
  private updateLotusParticles(count: number, focusLevel: number): void {
    const geometry = this.lotusParticles.geometry as THREE.BufferGeometry;
    const positions = new Float32Array(count * 3);
    
    for (let i = 0; i < count; i++) {
      // 专注度高时粒子更聚集，形成莲花形状
      const radius = 5 + Math.random() * (15 - focusLevel * 0.1);
      const angle = Math.random() * Math.PI * 2;
      const height = Math.random() * 10;
      
      positions[i * 3] = Math.cos(angle) * radius;
      positions[i * 3 + 1] = height;
      positions[i * 3 + 2] = Math.sin(angle) * radius;
    }
    
    geometry.setAttribute('position', new THREE.BufferAttribute(positions, 3));
  }
}
```

```typescript
/**
 * 禅修数据上链服务
 * 将验证过的脑波数据和冥想会话提交到现有Karma系统
 */
class MeditationKarmaService {
  private api: ApiPromise;
  
  /**
   * 提交冥想会话数据到Karma系统
   * @param sessionData 脑波数据摘要
   * @param metrics 冥想效果指标
   * @returns 交易哈希
   */
  async submitMeditationSession(
    sessionData: BrainwaveDataSummary,
    metrics: MeditationMetrics
  ): Promise<string> {
    // 数据压缩和隐私保护
    const compressedData = this.compressSessionData(sessionData);
    const privacyHash = this.generatePrivacyHash(compressedData);
    
    // 双Pallet流程：先提交冥想会话，再触发奖励计算
      const taskId = this.generateMeditationTaskId(sessionData);
      const brainwaveProof = this.generateBrainwaveProof(sessionData, metrics);
      const metricsDigest = this.compressMetrics(metrics);
      
      // 1. 提交冥想会话到Meditation Pallet
      const meditationExtrinsic = this.api.tx.meditation.submitSession(
        taskId,
        brainwaveProof,
        metricsDigest
      );
      
      // 2. 奖励计算由Reward Pallet的hook自动触发
      // 或手动调用：this.api.tx.reward.grantKarma(accountId, calculatedAmount, "Meditation")
    
    return await this.submitTransaction(meditationExtrinsic);
  }
  
  /**
   * 查询用户Karma福缘值余额
   * @param accountId 用户账户ID
   * @returns Karma余额
   */
  async getKarmaBalance(accountId: string): Promise<number> {
    const balance = await this.api.query.karma.karmaBalances(accountId);
    return balance.toNumber();
  }
  
  /**
   * 查询用户功德历史记录
   * @param accountId 用户账户ID
   * @returns 功德记录历史
   */
  async getMeditationKarmaHistory(accountId: string): Promise<any[]> {
     // 方式一：调用runtime视图函数（推荐，带分页）
     const recent = await (this.api.rpc as any).state.call(
       'Karma_getRecentMeritRecords',
       accountId,
       20 // 最近20条
     );
     const recentRecords = JSON.parse(recent.toString());

     // 方式二：直接查询存储（消费历史）
     const entries = await this.api.query.karma.meritConsumptionHistory.entries(accountId);
     const storageRecords = entries.map(([key, record]) => record.toJSON());

     return [...recentRecords, ...storageRecords].filter(r => r.action === 'Meditation');
   }
}
```

#### Karma福缘值激励设计
```typescript
/**
 * 基于冥想质量的Karma福缘值奖励计算
 * 结合会话时长、深度、一致性等多维度指标
 */
function calculateMeditationKarma(session: MeditationSession): number {
  const baseKarma = session.duration_minutes * 3; // 基础时长Karma奖励
  
  // 深度Karma奖励 (指数增长)
  const depthKarmaBonus = Math.pow(session.avg_meditation_depth / 100, 2) * 75;
  
  // 专注Karma奖励
  const focusKarmaBonus = (session.avg_focus_level / 100) * 45;
  
  // 一致性Karma奖励 (数据质量)
  const qualityKarmaBonus = (session.quality_score / 100) * 30;
  
  // 连续修行Karma奖励
  const streakMultiplier = 1 + (session.consecutive_days * 0.15);
  
  const totalKarma = (baseKarma + depthKarmaBonus + focusKarmaBonus + qualityKarmaBonus) * streakMultiplier;
  
  return Math.round(totalKarma);
}

/**
 * 冥想Karma奖励原因枚举
 * 扩展现有Karma系统的奖励类型
 */
enum MeditationKarmaReason {
  BasicMeditation = "BasicMeditation",     // 基础冥想
  DeepMeditation = "DeepMeditation",       // 深度冥想
  FocusedPractice = "FocusedPractice",     // 专注修行
  MindfulnessSession = "MindfulnessSession", // 正念会话
  LovingKindness = "LovingKindness"        // 慈悲冥想
}
```
